# Задание 1

## Задача 1

### Текст задачи

16. Сформировать список из названий дней недели.

### Алгоритм решения

Реализация:
Ввод: номера дней недели 1-7
Вывод: соответствующие номерам дни недели

- Делаем глобальный список дней недели
- Получаем из консоли числа k - количество желаемых дней недели и номера дней недели (с проверками на то подходят они нам или нет)
- Формируем список дней недели по полученным данным
- Выводим его в консоль

## Задача 2

### Текст задачи

16. Возвратить true, если в числе есть цифра k, и false — в противном случае

### Алгоритм решения

- Получаем из консоли 2 числа: целое (в котором будем искать цифру) и цифру
- Проверяем вводимые данные на корректность
- Рекурсивно проходим по числу, смотрим есть ли там нужная цифра
- Выводим результат изысканий (есть / нет)
