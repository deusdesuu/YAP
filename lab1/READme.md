## Задача 3

### Текст задачи

2. Создайте собственные функции для выполнения основных операций над списками (добавление/
удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

Реализация:
- AddFront() - добавление в начало: обычное добавление элемента через изменение головного элемента списка
- AddBack() - добавление в конец: конкатенация исходного списка со списком состоящим из одного нового элемента [elem]
- PopAt() - удаление по индексу: копирование списка без элемента с указанным индексом
- Search() - поиск всех вхождений элемента: проходим по списку, если элемент совпадает с искомым, добавляем найденный индекс в список
- Concat() - конкатенация: стандартная конкатенация
- GetAt() - получение элемента по номеру: вывод элемента по индексу
