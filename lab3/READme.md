# Задание 1

## Задача 1

### Текст задачи

16. На основе списка, содержащего двоичные значения цифр от 1 до 9, получить их
десятичные представления. (Seq)

### Алгоритм решения

Реализация:
Ввод: кол-во элементов n, элементы формата 1-1001
Вывод: список элементов 1-9

- Получаем число элементов n
- Получаем строку из консоли
- Проверяем, что она состоит только из 0 и 1
- Проверяем, что введенное число из 1..9
- Формируем множество
- Конвертируем bin->int при помощи Seq.map

## Задача 2

### Текст задачи

16. Список содержит символы, обозначающие шестнадцатеричные цифры.
Получить шестнадцатеричное число из этих цифр. (Seq)

### Алгоритм решения

Реализация:
Ввод: кол-во элементов n, элементы формата 0-F
Вывод: шестнадцатеричное число

- Получаем число элементов n
- Получаем char из консоли (при введении >1 символа берется только первый)
- Проверяем что полученный char из 0..9A..F
- Формируем множество
- Собираем число воедино при помощи Seq.fold
